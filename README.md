# 🤖 Chatbot Pro

A full-stack SaaS AI chatbot platform that allows users to create, customize, and embed intelligent chatbots into their websites.

---

## 🚀 Features

- 🔐 Authentication (Google OAuth with Firebase)
- 🤖 AI-powered real-time chat
- 🎨 Bot customization (name, color, greeting)
- 📊 Analytics dashboard (chat tracking & usage stats)
- 💬 Embeddable chat widget
- 🧑‍💼 Admin panel for managing users & sessions
- 💳 Stripe integration (optional billing)

---

## 🛠️ Tech Stack

- Frontend: Next.js, React, Tailwind CSS  
- Backend: Next.js API Routes  
- Database: Firebase Firestore  
- Auth: Firebase Authentication  
- AI: OpenAI API  
- Deployment: Vercel  

---

npm run dev
